# 🔑 Login & Signup Form

> A clean and modern authentication form built with **React** and **Next.js**.  
> This project focuses on **live form validation**

---

## ✨ Key Features

- 📝 **Real-time Validation** – Immediate feedback on input fields for better user experience  
- 🔐 **Secure Authentication** – Separate login and signup flows  
- 🎨 **Responsive Design** – Works seamlessly on desktop and mobile devices
